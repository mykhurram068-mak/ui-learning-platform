📱 Featured snippet
Responsive navbar with mobile hamburger menu – pure HTML/CSS/JS
🔍 Live preview (resize your browser or open on phone)
📖 How it works

This navbar uses flexbox for alignment, a media query for mobile (max-width: 768px), and a few lines of JavaScript to toggle the mobile menu class.

💻 Complete code (copy & paste)
<!-- Responsive navbar HTML/CSS/JS -->
<nav style="background:#1e293b; padding:1rem 2rem;">
  <div style="max-width:1200px; margin:0 auto; display:flex; justify-content:space-between; align-items:center;">
    <a href="#" style="color:white; font-size:1.5rem; font-weight:bold;">Logo</a>
    <button class="hamburger" style="display:none; background:none; border:none; cursor:pointer;">
      <span style="display:block; width:25px; height:3px; background:white; margin:5px 0;"></span>
      <span style="display:block; width:25px; height:3px; background:white; margin:5px 0;"></span>
      <span style="display:block; width:25px; height:3px; background:white; margin:5px 0;"></span>
    </button>
    <ul class="nav-menu" style="display:flex; gap:2rem; list-style:none;">
      <li><a href="#" style="color:#cbd5e1;">Home</a></li>
      <li><a href="#" style="color:#cbd5e1;">Snippets</a></li>
    </ul>
  </div>
</nav>

<style>
  @media (max-width: 768px) {
    .hamburger { display: block !important; }
    .nav-menu {
      position: fixed; left: -100%; top: 70px;
      flex-direction: column; background: #1e293b;
      width: 100%; transition: 0.3s;
    }
    .nav-menu.active { left: 0; }
  }
</style>

<script>
  document.querySelector('.hamburger').addEventListener('click', () => {
    document.querySelector('.nav-menu').classList.toggle('active');
  });
</script>
