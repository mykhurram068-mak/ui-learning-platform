📱 Featured snippet
Responsive navbar with mobile hamburger menu – pure HTML/CSS/JS (FIXED)
🔍 Live preview – resize browser or open on phone → tap the ☰ icon
📖 How it works

This navbar uses flexbox for alignment, a media query (max-width: 768px) for mobile, and JavaScript to toggle the .active class on the mobile menu.

💻 Complete code (copy & paste)
<!-- HTML -->
<nav class="navbar">
  <div class="nav-container">
    <a href="#" class="logo">Logo</a>
    <button class="hamburger" id="hamburgerBtn">
      <span></span><span></span><span></span>
    </button>
    <ul class="nav-menu" id="navMenuList">
      <li><a href="#">Home</a></li>
      <li><a href="#">Snippets</a></li>
    </ul>
  </div>
</nav>

<!-- CSS (add to <style>) -->
.navbar { background: #1e293b; padding: 1rem 2rem; }
.nav-container { max-width: 1200px; margin: 0 auto; display: flex; justify-content: space-between; align-items: center; }
.logo { color: white; font-size: 1.5rem; text-decoration: none; }
.nav-menu { display: flex; gap: 2rem; list-style: none; }
.hamburger { display: none; background: none; border: none; cursor: pointer; }
.hamburger span { display: block; width: 25px; height: 3px; background: white; margin: 5px 0; }

@media (max-width: 768px) {
  .hamburger { display: block; }
  .nav-menu {
    position: fixed; left: -100%; top: 70px; flex-direction: column;
    background: #1e293b; width: 100%; transition: 0.3s; padding: 2rem 0; gap: 1.5rem;
  }
  .nav-menu.active { left: 0; }
}

<!-- JavaScript -->
const hamburger = document.getElementById('hamburgerBtn');
const navMenu = document.getElementById('navMenuList');

hamburger.addEventListener('click', () => {
  navMenu.classList.toggle('active');
});

// Close menu when clicking a link
document.querySelectorAll('.nav-menu a').forEach(link => {
  link.addEventListener('click', () => {
    navMenu.classList.remove('active');
  });
});
