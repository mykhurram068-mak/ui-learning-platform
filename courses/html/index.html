Why Learn HTML?

HTML (HyperText Markup Language) is the foundation of every website. It's the first language you need to master for web development. This free course will teach you HTML and basic CSS step by step, with hands-on examples you can try immediately.

What You'll Learn:

  • Lesson 1.1: Building Your First Header – headings, paragraphs, and document structure.
  • Lesson 1.2: Links & Navigation – connect pages together with hyperlinks.
  • Lesson 1.3: Images & Multimedia – add images, videos, and audio to your pages.
  • Lesson 1.4: Lists & Tables – organise data with ordered lists, unordered lists, and tables.
  • Lesson 1.5: Forms & Inputs – build interactive forms to collect user data.

Each lesson includes live code examples and practice challenges. Start with Lesson 1.1 below.
